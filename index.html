Typography

Type Scale

This typography follows a major third (1.250) scale on a base font size 14px. Headers use IBM Plex Mono, medium. It’s a neutral, yet friendly Grotesque style with great legibility.

Body and title

The main body is using 14px, regular Noto Sans. The Noto family is designed with the goal of achieving visual harmony across multiple languages/scripts.
